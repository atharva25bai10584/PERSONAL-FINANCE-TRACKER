##  Overview of the Project
The **Personal Finance Tracker** is a simple and user-friendly desktop application built using Python and Tkinter. It allows users to efficiently manage their income and expenses, categorize financial transactions, analyze spending patterns, and generate monthly and category-based reports.  
The application supports importing and exporting transaction data in CSV format, making backups and data transfers convenient.

---

##  Features
- Add, update, delete financial transactions
- Track **Income & Expenses** with date, category, and description
- Custom category creation
- **Monthly summary reports** with Income, Expense, and Net Balance
- **Category-wise report** for expense analytics
- Import and Export transactions as **CSV**
- Persistent data storage using **JSON**
- Intuitive GUI built with Tkinter

##  Steps to Install & Run the Project
### **Prerequisites**
- Python 3.x installed on system

### **Running Instructions**
```bash
# Step 1: Download/clone the project folder
cd path/to/project

# Step 2: Run the script
python finance_tracker.py
```
Once executed, the application window will open automatically.
Instructions for Testing
