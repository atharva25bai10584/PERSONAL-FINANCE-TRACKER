# Problem Statement

A personal finance tracker application that enables users to record,
manage, and review their income and expenses through a simple desktop
GUI built using Tkinter.

# Scope of the Project

-   Allow users to add, update, and delete financial transactions.
-   Support categorization of expenses and incomes.
-   Provide CSV import and export functionality.
-   Generate monthly and category-wise summary reports.
-   Persist data locally using JSON storage.

# Target Users

-   Individuals who want a lightweight, offline personal finance
    tracking tool.
-   Users who prefer a simple desktop application without cloud
    dependency.
-   Anyone who needs basic financial reporting (monthly/category
    summaries).

# High-Level Features

-   Add, edit, delete transactions with fields: date, type, amount,
    category, description.
-   Custom category creation.
-   View transactions in a sortable table.
-   Import and export CSV files.
-   Monthly and category summary reports.
-   JSON-based persistent storage.
